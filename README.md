### Honest tab

A digital POS/tab system for small businesses.

#### Install requirements and activate virtual environment

In a bash shell run `. requirements.bash`.

#### Run locally

1. `reflex run`
2. Open [http://localhost:3000/](http://localhost:3000/) in a browser.
Also checkout the `/admin` subpage.

#### Setting up a backend (google service account)

Without a backend not much exciting will happen. To setup one:
1. Create a copy of
[this sheet](https://docs.google.com/spreadsheets/d/1get0Mcr_Pso1B3Nn-VtQV4gqKDYVcn9-zus1u1Xddh8/edit?usp=sharing)
(requires a google account).
2. Follow steps 1-7 of
[this guide](https://docs.gspread.org/en/latest/oauth2.html#for-bots-using-service-account).
